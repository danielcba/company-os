# Context

Version: 1.0
Status: Official
Owner: Company OS

---

# Definition

Context is the cognitive interpretation of evidence through the mental model that provides the highest explanatory coherence relative to a specific purpose.

Context does not create meaning.

It reveals the most coherent meaning supported by the available evidence.

---

# Purpose

Context transforms organized evidence into understandable knowledge.

Its purpose is not to explain reality, but to establish the most coherent interpretation from which reasoning can continue.

---

# Why It Matters

Evidence alone does not explain reality.

Different intelligences may organize exactly the same evidence while reaching different interpretations because they possess different knowledge, experiences, objectives, and mental models.

Context makes reasoning possible by selecting the interpretation that best fits the available evidence and the current purpose.

Without Context, intelligence cannot distinguish between multiple plausible explanations.

---

# Cognitive Contract

## Input

- Organized Evidence
- Mental Models
- Purpose

## Transformation

Interpret the available evidence through the mental model that achieves the highest explanatory coherence relative to the current purpose.

## Output

An Active Context that becomes the foundation for pattern recognition and subsequent reasoning.

# Examples

## Valley

Three professionals observe the same valley.

The geologist interprets geological structures.

The military officer identifies strategic terrain.

The farmer evaluates soil, sunlight, and water availability.

The evidence remains identical.

Only the active Context changes because different mental models become active.

---

## Medical Diagnosis

Evidence:

- Chest pain
- Sweating
- Hypertension
- Age: 60

Several medical mental models become active simultaneously.

The model with the highest explanatory coherence emerges as the active Context.

If later evidence contradicts that interpretation, another model may become active.

---

# Non-Examples

Context is not opinion.

Opinion may exist without evidence.

Context cannot.

---

Context is not evidence.

Evidence organizes reality.

Context interprets it.

---

Context is not imagination.

Every valid Context must remain coherent with the available evidence.

---

# Design Implications

Company OS never generates Context directly.

It activates multiple compatible mental models.

Those models compete according to explanatory coherence.

The model providing the strongest coherent explanation becomes the Active Context.

Reasoning continues from that Context.

This architecture allows multiple valid interpretations while preventing arbitrary conclusions.
